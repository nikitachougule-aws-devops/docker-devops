## Installation on Windows

## 🔹 Prerequisites

Before installing Docker, make sure:

- 64-bit Windows 10/11  
- Virtualization enabled in BIOS  
- WSL2 feature enabled  

---

## 🔹 Step 1: Enable WSL (Windows)
Open PowerShell as Administrator and run:
```bash
wsl --install
```
👉 Then restart your system.

---

## 🔹 Step 2: Install Docker Desktop
- Go to the Official Docker Website
🔗 [Get Started with Docker](https://www.docker.com/get-started)
- Download Docker Desktop for Windows
- Run the installer
- During installation: Select Use WSL 2 instead of Hyper-V

---

## 🔹 Step 3: Start Docker Desktop
- Open Docker Desktop from Start Menu
- Wait until it shows: Docker is running

---

## 🔹 Step 4: Verify Installation
- Open Command Prompt or PowerShell:
```bash
docker --version
```
---

## 🔹 Step 5: Run First Container
```bash
docker run hello-world
```
👉 This will:
- Download image from Docker Hub
- Run a test container
- Show success message
---

## 🔹 Step 6: Check Running Containers
```bash
docker ps
```
