## Basic Docker Commands

### Check Docker Version
docker --version

### View Docker Info
docker info

## Working with Images

### List Images
docker images

### Pull Image from Docker Hub
docker pull nginx

### Remove Image
docker rmi <image_id>

## Working with Containers

### Run a Container
docker run nginx

### Run Container in Background (Detached Mode)
docker run -d nginx

### Run Container with Name
docker run -d --name mynginx nginx

### List Running Containers
docker ps

### List All Containers (Including Stopped)
docker ps -a

### Stop a Container
docker stop <container_id>

### Start a Container
docker start <container_id>

### Remove a Container
docker rm <container_id>

## Inspect & Logs

### View Logs
docker logs <container_id>

### Inspect Container Details
docker inspect <container_id>

## Execute Commands Inside Container

###  Access Container Terminal
docker exec -it <container_id> bash

## Port Mapping

###  Run Container with Port Mapping
docker run -d -p 8080:80 nginx

👉 Access in browser: http://localhost:8080

## Volume Management

### Create Volume
docker volume create myvolume

### Use Volume
docker run -d -v myvolume:/data nginx

## Image Build (Dockerfile)

### Build Image
docker build -t myapp .

## Push & Pull

### Login to Docker Hub
docker login

### Push Image
docker push <username>/myapp

### Pull Image
docker pull <username>/myapp

## Cleanup Commands

### Remove All Stopped Containers
docker container prune

### Remove Unused Images
docker image prune

### Remove Everything
docker system prune -a

## Real-Time Workflow Example

### Pull image
docker pull nginx

### Run container
docker run -d -p 8080:80 --name web nginx

### Check running containers
docker ps

### Stop container
docker stop web

### Remove container
docker rm web

## Summary

- Images are templates used to create containers
- Containers are running instances of images
- Use docker run to start containers
- Use docker ps to check status
- Use docker stop & docker rm to manage containers
- Use docker build to create custom images
